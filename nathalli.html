HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Abra o Visual Studio Code clicando em seu ícone. Ele deverá estar no menu "Iniciar" (Windows), ou na pasta "Aplicativos" (Mac), ou no menu Apps, no Linux.
